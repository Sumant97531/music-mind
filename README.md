# Music Mind 🎧

> **"Shape your mind. One sound at a time."**

A hybrid music recommendation system combining content-based filtering,
collaborative filtering, and mood-aware inference to deliver explainable,
wellness-oriented recommendations.

## Quick Start

### 1. Clone and install

```bash
git clone https://github.com/Sumant97531/music-mind.git
cd music_mind

python -m venv venv
venv\Scripts\activate          # Windows
# source venv/bin/activate     # Mac / Linux

pip install -r requirements.txt
```

### 2. Add your data

Place the raw CSVs in `data/raw/`:
```
data/raw/Music Info.csv
data/raw/User Listening History.csv
```

### 3. Build the pipeline (run once)

```bash
python scripts/build_pipeline.py
```

This runs 4 steps in order:
1. Clean raw data → `data/processed/cleaned_data.csv`
2. Train transformer → `models/transformer.joblib` + `data/processed/transformed_data.npz`
3. Build interaction matrix → `data/processed/interaction_matrix.npz` + `collab_filtered_data.csv`
4. Transform collab-filtered data → `data/processed/transformed_hybrid_data.npz`

### 4. Run the app

**Streamlit UI (recommended):**
```bash
streamlit run app/streamlit_app.py
```
Open: http://localhost:8501

**FastAPI (for deployment / API use):**
```bash
uvicorn app.main:app --reload
```
Docs: http://localhost:8000/docs

## How It Works

### Recommendation Score

```
Score = w_content × content_sim
      + w_collab  × collab_sim
      + w_mood    × mood_sim      ← only when mood is provided
```

### Explainability

No SHAP. Fast feature deviation instead:
```
importance(f) = (rec_value - seed_value) / std(f)
```
< 1 ms per song, human-readable, stable output.

### Mood Profiles

29 predefined mood/activity states (study, gym, heartbreak, coding, road trip…).
Free-text input is resolved via alias matching → mood profile → seed song.
